Usage:
  nixfromnpm <packagename> <outputpath>

Options:
  -r, --registries REGISTRIES...    Use only these registries
  --extra-registries REGISTRIES..   Use these in addition to default registry
  --no-cache                        Don't use existing cache
  --timeout SECONDS                 Seconds after which to fail fetching a 
                                    package
  --node-version VERSION            Use this version of node
